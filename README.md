# OCR Web Application

This project is an AI-based Optical Character Recognition (OCR) system that extracts text from images using Tesseract.

## 🔧 Features
- Upload image
- Extract text from image
- Simple web interface
- Multi-language support

## 🛠️ Technologies Used
- Python
- Flask
- Tesseract OCR
- HTML, CSS

## 🚀 How to Run
1. Install dependencies
2. Run app.py
3. Open browser at localhost

## 📌 Project Type
Mini Project (Semester 4)
